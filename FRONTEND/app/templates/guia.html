Guía de Expresiones Matemáticas

Esta guía te ayudará a entender cómo ingresar correctamente las expresiones matemáticas en nuestra página.

Símbolos y Operadores Básicos

Operación Expresión Ejemplo
Potencia ** x**2 (x al cuadrado)
Raíz Cuadrada sqrt() sqrt(x)
Logaritmo Natural log() log(x)
Logaritmo Base 10 log10() log10(x)
Exponencial exp() exp(x) (e^x)
Valor Absoluto abs() abs(x)
Raíz Enésima x**(1/n) x**(1/3) (raíz cúbica)
Módulo % x % 2 (residuo de x entre 2)

Funciones Trigonométricas

Función Expresión Ejemplo
Seno sin() sin(x)
Coseno cos() cos(x)
Tangente tan() tan(x)
Arcoseno asin() asin(x)
Arcocoseno acos() acos(x)
Arcotangente atan() atan(x)
Seno Hiperbólico sinh() sinh(x)
Coseno Hiperbólico cosh() cosh(x)
Tangente Hiperbólica tanh() tanh(x)

Funciones Especiales

Función Expresión Ejemplo
Signo sign() sign(x) (devuelve -1, 0 o 1)
Máximo max() max(a, b)
Mínimo min() min(a, b)
Factorial factorial() factorial(5) (devuelve 120)
Función de Paso step() step(x) (devuelve 0 si x < 0, 1 si x ≥ 0)
